# KOTSF — Midjourney Icon Prompts (copy-paste, one per icon)

Each line below is a **complete, self-contained Midjourney prompt** — paste as-is.

**For a consistent set:** generate **#1 the Sacred Flame first**, upscale the best, copy its image URL, and
append `--sref <that-url>` to every other prompt so the whole set shares one look. Keeping a fixed `--seed`
(e.g. `--seed 777`) on all of them helps too. Style target: Aion Rising badges × Mad Monkies grunge ×
forgottenrunes.com occult-folk. `--style raw` keeps it graphic; `--v 6` (bump to `--v 7` if you're on it).
