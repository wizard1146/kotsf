Type study

Barlow — the one family

Body · Semi Condensed display · mono numerals. Weight + width carry the whole hierarchy.

Why Barlow

Barlow is a low-contrast grotesque with a faint industrial/condensed feel — warm enough for firelit fantasy, sturdy enough for dense UI, and it ships a full weight range (300–800) plus true Semi Condensed and Condensed cuts. That range means one family can do the whole game: condensed heavy for epic titles, regular for prose, mono for the numbers. No second typeface needed.

Display — Barlow Semi Condensed (titles)

Titles use the Semi Condensed cut at 700. It packs more character per line (good for a long game name) and reads more "epic/game" than the regular width.

The numbers stay mono

Pressures, counters, and contest values use a monospaced face — alignment beats family here. Barlow proportional would make the readout jitter as digits change.

Recommendation

Ship Barlow as the single game family. Three roles:

• Barlow Semi Condensed 700 — titles & screen headings (--display)
• Barlow 400/500 — all prose, UI, buttons (--sans)
• system mono — every number (--mono)

Optional: Barlow Condensed 700 for very long/dramatic end-screen titles. Before the desktop build, self-host the woff2 so the game runs offline (currently loaded via Google Fonts <link>).
